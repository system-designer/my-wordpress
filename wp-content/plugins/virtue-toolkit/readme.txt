Custom Portfolio and Shortcode functionality for Virtue and Pinnacle Wordpress Theme

== Description ==

Custom Portfolio and Shortcode functionality for Virtue and Pinnacle Wordpress Theme

Not intended to work with any other themes.

== Installation ==

Install the plugin into the `/wp-content/plugins/` folder, and activate it.
